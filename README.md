# jokify

**jokify** is a minimal command‑line utility written in Python that pulls a random joke from the [JokeAPI](https://v2.jokeapi.dev/) and displays it.

## Features
- One‑liner execution
- Supports both single‑line and two‑part jokes
- Zero external dependencies beyond the Python standard library (uses `urllib` for HTTP)

## Installation
```bash
# Clone the repo (or copy the single file)
git clone https://github.com/yourname/jokify.git
cd jokify
```

## Usage
```bash
python jokify.py
```

You will see a random joke printed to the console.
